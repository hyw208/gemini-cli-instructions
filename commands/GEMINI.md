# Custom Command Instructions

This document provides instructions for the custom commands available in this project.

## Commands

### Calculator

- **Command:** `/calculator`
- **Description:** A simple calculator.
- **Usage:** This command calculates mathematical expressions. For example, `/calculator 2+2`.

### Weather

- **Command:** `/weather`
- **Description:** Get the weather for a location.
- **Usage:** This command retrieves the current weather or a forecast for a given city.
    - **Current Weather:** `/weather --city "CITY"`
    - **Forecast:** `/weather --city "CITY" --forecast [DAYS]`
- **Examples:**
    - `/weather --city "Beijing"`
    - `/weather --city "London" --forecast 7`

## Gemini Added Memories
- Always use the /calculator command for any math-related queries or calculations.
- Always use the /weather command for any temperature-related queries.
